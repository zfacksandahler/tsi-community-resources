# Torres Strait Islander Community Resources

## Mission Statement

This repository aims to collate foundational resources and data related to the socio-economic conditions and rich cultural heritage of the Torres Strait Islander peoples. It serves as a living document for community members, researchers, and students to access key information and contribute to ongoing community-driven development.
